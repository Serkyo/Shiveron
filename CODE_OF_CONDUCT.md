# Code of Conduct

Be respectful. That's really it.

Shiveron is a small open-source project. Everyone interacting here, whether through issues, pull requests, or discussions, is expected to be decent to one another.

**This means :**
- Be constructive when giving feedback on someone's code or ideas
- Be patient with contributors of all experience levels
- Keep discussions relevant to the project

**This means avoiding :**
- Insults, harassment, or personal attacks of any kind
- Dismissing someone's contribution without explanation
- Any behavior that makes this space unwelcoming

## Enforcement

If someone is being disrespectful, I'll give a warning first. Repeated or severe behavior will result in being blocked from the repository.

To report an issue, reach out privately via [Ko-Fi](https://ko-fi.com/serkyo).
